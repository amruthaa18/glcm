# GLCM Texture Feature Extraction

This project performs image texture analysis using Gray Level Co-occurrence Matrix (GLCM) techniques in Computer Vision and Image Processing.

## Features
- Texture feature extraction
- GLCM matrix computation
- Image analysis and visualization
- Statistical feature calculation
- Pattern recognition concepts

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Scikit-image
- Google Colab

## Concepts Covered
- Computer Vision
- Image Processing
- Texture Analysis
- Feature Extraction
- Pattern Recognition

## Project Goal
The main objective of this project is to analyze image textures using GLCM-based statistical methods for computer vision applications.
